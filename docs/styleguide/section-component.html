2.1 #Component - Carousel Carousel

Selector: .carousel, .carousel__image, .carousel__commands, .carousel__button, .carousel__current-image

Image is full size in the component, commands wrapper has absolute position and full size also.
Buttons and curent image indicator are flex.

Source: src/components/Carousel/Carousel.scss, line 1

2.2 #Component - Dropdown Dropdown

Selector: .dropdown, .dropdown__header, .dropdown__title, .dropdown__icon, .dropdown__icon--open, .dropdown__body, .dropdown__body--open

2 parts with a vertical flex: header and body.
Header is horizontal flex with a title on the left and an icon on the right. Icon is rotated 180deg on dropdown opening.
Body contains either a list or a paragraph. There is a fade animation on opening/closing.
Dropdown opening is managed by turning the body's opacity, line-height and font-size from zero to final size. There is a shift in animation delay to have it fluent.
Body is offset to the top in order to cover the radius at the bottom of the header. A margin and padding are used to compensate the offset.

Source: src/components/Dropdown/Dropdown.scss, line 3

2.3 #Component - Dropdown wrapper Dropdown wrapper

Selector: .dropdown-wrapper, .dropdown-wrapper--large-screen-2-col

Used to manage the dropdown display : vertical flex in most cases.
If class modifier &--large-screen-2-colis used : the dropdown will be in horizontal flex display on large screen.
In this case the two dropdowns have the same height while open, due to the align-items stretch property.

Source: src/components/DropdownWrapper/DropdownWrapper.scss, line 1

2.4 #Component - Footer Footer

Selector: .footer, .footer__wrapper, .footer__logo, .footer__disclaimer

There is a wrapper to make the background color to be 100% of the screen width (.container is on .footer and not .footer__wrapper).
Everything is centered. Line height of the disclaimer is fixed, so changing the font-size does not change the height of the box.

2.6 #Component - Header Header

Selector: .header, .header__logo, .header__ul, .header__li, .header__link, .header__link--active

Horizontal flex with logo on the left and nav on the right.
The nav contain a list of 2 links, the one corresponding to curent page is underlined.
Display inline-block and big line-height of the li/a allow a large clickable surface.

Source: src/components/Header/Header.scss, line 3

2.7 #Component - Hero Hero

Selector: .hero, .hero--fixed-height

It has a background image added with inline CSS. a mask can also be added with inline CSS depending on props (black with 30% opacity).
Inner text is on the left for small screen and centered for large one.
Depending on props the height can be fixed whatever the screen with the class modifier &--fixed-height.
